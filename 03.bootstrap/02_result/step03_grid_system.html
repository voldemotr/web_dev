컨테이너(container)


컨테이너 (div.container로 생성)
extra-small(576px 미만) : 가로 100%
small(576px 이상 ~ 768px 미만) : 가로 540px
medium(768px 이상 ~ 992px 미만) : 가로 720px
large(992px 이상 ~ 1200px 미만) : 가로 960px
extar-large(1200px 이상) : 가로 1140px
+ 가운데 정렬 (margin:auto)

컨테이너-플루이드 (div.container-fluid로 생성) 중단점과 관계 없이 100%로 표시

div.container-sm (=container)

div.container-md (md 전까지는 fluid)

div.container-lg (lg 전까지는 fluid)

div.container-xl (xl 전까지는 fluid)
p-3 : 1rem(기본 16px)만큼 전체 방향으로 padding
my-3 : 1rem만큼 y축 (위,아래)로 margin
(위=t, 오른쪽=r, 왼쪽=l, 아래=b, x=왼쪽+오른쪽, y=위+아래) border : 1px solid grey의 구분선
bg-dark : 검정색 배경색
text-white : 흰색 글씨색
bg-primary : 파란색 배경색 (primary, secondary, ...)
